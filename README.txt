ATLANTIC SURF SCHOOL - SITE WEB

STRUCTURE :
- index.html : Page principale
- style.css : Feuilles de style
- script.js : JavaScript
- images/ : Dossier pour les images

INSTRUCTIONS POUR VISUAL STUDIO CODE :

1. Créer un nouveau dossier "surf-school-maroc"

2. Dans ce dossier, créer 4 fichiers :
   - index.html
   - style.css
   - script.js
   - README.txt

3. Créer un dossier "images" (vide pour l'instant)

4. Copier le contenu de chaque fichier dans les fichiers correspondants

5. Pour tester le site :
   - Ouvrir index.html dans un navigateur
   - OU dans VS Code, installer l'extension "Live Server" et cliquer sur "Go Live"

IMAGES :
Le site utilise des images depuis Unsplash en ligne.
Pour utiliser vos propres images :
1. Télécharger des images dans le dossier "images"
2. Modifier les références dans le code HTML

FONCTIONNALITÉS :
✓ Design responsive (mobile/tablette/desktop)
✓ Formulaire d'inscription avec validation
✓ Formulaire de contact
✓ Galerie interactive
✓ Animations au défilement
✓ Menu mobile
✓ Bouton retour en haut

PERSONNALISATION :
- Modifier les couleurs dans style.css (variables :root)
- Changer les tarifs dans index.html
- Ajouter vos propres images
- Modifier les textes et descriptions

Le site est prêt à être déployé sur un hébergement web.
